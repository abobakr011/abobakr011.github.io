📘 Python

📖 مقدمة

بايثون لغة برمجة عالية المستوى، سهلة القراءة، تُستخدم في تطوير الويب، تحليل البيانات، الذكاء الاصطناعي، الأتمتة، وأكثر.

🧮 المتغيرات والطباعة

name = "أحمد"
age = 25
print("الاسم:", name)
print("العمر:", age)

يتم تخزين بيانات في متغيرات، وطباعتها على الشاشة.

🔁 الحلقات

for i in range(5):
    print(i)

تستخدم لطباعة الأرقام من 0 إلى 4.

🧠 الشروط

x = 10
if x > 5:
    print("x أكبر من 5")
else:
    print("x أقل أو يساوي 5")

تنفيذ كود بناءً على شرط.

📦 الدوال

def جمع(a, b):
    return a + b

print(جمع(3, 5))

الدالة تعيد ناتج جمع رقمين.

📘 Assembly

📖 مقدمة

Assembly هي لغة منخفضة المستوى تعتمد على معمارية المعالج، وتستخدم للتعامل المباشر مع الهاردوير.

🔧 طباعة رسالة باستخدام NASM

section .data
  msg db "Hello, World!",0Ah
  len equ $ - msg

section .text
  global _start

_start:
  mov eax, 4      
  mov ebx, 1      
  mov ecx, msg    
  mov edx, len    
  int 0x80        

  mov eax, 1      
  xor ebx, ebx    
  int 0x80

الشرح: يتم استدعاء دالة النظام sys_write لطباعة الرسالة، ثم sys_exit لإنهاء البرنامج.

📘 الذكاء الاصطناعي

📖 مقدمة

الذكاء الاصطناعي (AI) هو مجال يهدف إلى تطوير أنظمة ذكية قادرة على محاكاة التفكير البشري.

🧠 مثال باستخدام Python

from sklearn.tree import DecisionTreeClassifier

X = [[0, 0], [1, 1]]
y = [0, 1]

clf = DecisionTreeClassifier()
clf.fit(X, y)

print(clf.predict([[2, 2]]))

الشرح: نستخدم خوارزمية الشجرة لاتخاذ قرار بناءً على البيانات.

📘 تعلم الآلة

📖 مقدمة

تعلم الآلة (Machine Learning) يتيح للبرامج التعلم من البيانات دون برمجة صريحة.

🔧 تصنيف بيانات

from sklearn.linear_model import LogisticRegression

X = [[0], [1], [2], [3]]
y = [0, 0, 1, 1]

clf = LogisticRegression()
clf.fit(X, y)

print(clf.predict([[1.5]]))

الشرح: النموذج يتعلم العلاقة بين المدخلات والمخرجات ثم يتنبأ بقيمة جديدة.
