📘 الأمن السيبراني

📖 مقدمة

الأمن السيبراني يهدف لحماية الأنظمة والمعلومات من التهديدات الرقمية مثل الـ hacking والاختراق.

🔐 مثال تشفير قيصر بسيط

def caesar(text,shift):
  result=""
  for c in text:
    if c.isalpha():
      result+=chr((ord(c)-ord('أ')+shift)%28+ord('أ'))
    else: result+=c
  return result

print(caesar("السلام",3))

تشفير بسيط بتحويل حروف بمقدار ثابت (shift).

📌 خلاصة

  • فهم مبادئ التشفير أساس لحماية الأنظمة.
  • مهم لتطوير أدوات مثل Nmap، Firewalls، IDS/IPS.
