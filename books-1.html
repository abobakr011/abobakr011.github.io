📖 المقدمة

Python هي واحدة من أكثر لغات البرمجة استخدامًا اليوم، وتُعرف بسهولة تعلمها ومرونتها. تُستخدم في تطوير الويب، تحليل البيانات، الذكاء الاصطناعي، والتطبيقات المختلفة.

في هذا الكتاب، سنأخذك من المبتدئ حتى المستوى المتقدم بأسلوب عملي واضح.

🔧 الأساسيات

  • 🔹 التعرف على المتغيرات وأنواع البيانات
  • 🔹 استخدام الجمل الشرطية (if/else)
  • 🔹 الحلقات (for / while)
  • 🔹 الدوال (Functions)
  • 🔹 القوائم Lists والتعامل معها
  • 🔹 القواميس Dictionaries
  • 🔹 المجموعات Sets والـ Tuples
x = 10
if x > 5:
    print("العدد أكبر من 5")

🚀 التطبيقات

في هذا الفصل، سنقوم بعمل مشروع بسيط: برنامج آلة حاسبة.

def calculator():
    op = input("اختر العملية (+ - * /): ")
    x = float(input("العدد الأول: "))
    y = float(input("العدد الثاني: "))
    if op == '+':
        print(x + y)
    elif op == '-':
        print(x - y)
    elif op == '*':
        print(x * y)
    elif op == '/':
        print(x / y)
calculator()

  • 🔹 الأحداث (Events)
  • 🔹 التعامل مع DOM (وثيقة HTML)
let x = 10;
if (x > 5) {
  console.log("العدد أكبر من 5");
}

📖 المقدمة

🧠 شرح شامل للغة Java

الكائنات (Objects) والفئات (Classes)

Java هي لغة كائنية التوجه بالكامل. كل شيء يُعبّر عنه ككائن. الفئات هي القوالب التي تُنشأ منها الكائنات.

الوراثة (Inheritance)

تتيح Java إنشاء تسلسل هرمي من الكائنات، مما يسمح بإعادة استخدام الكود.

الواجهات (Interfaces) والتجريد (Abstraction)

يمكنك استخدام الواجهات لتعريف مجموعة من الوظائف التي يجب على الفئة أن تنفذها.

إدارة الذاكرة

Java تُدير الذاكرة باستخدام جامع القمامة (Garbage Collector) الذي يحرر الذاكرة تلقائيًا.

الاستثناءات (Exceptions)

توفر Java آلية قوية للتعامل مع الأخطاء من خلال try-catch-finally.

المكتبات (Libraries)

Java تملك مكتبة ضخمة تشمل أدوات للعمل مع الملفات، الشبكات، الواجهات الرسومية، قواعد البيانات، والمزيد.

🧠 شرح شامل للغة C++

البرمجة الكائنية (OOP)

توفر C++ مفاهيم قوية مثل الكائنات (Objects)، الأصناف (Classes)، الوراثة (Inheritance)، التغليف (Encapsulation)، وتعدد الأشكال (Polymorphism).

المؤشرات والمراجع

تدعم المؤشرات كما في C، بالإضافة إلى المراجع (References) لتسهيل التعامل مع الكائنات في الذاكرة.

القوالب (Templates)

تمكن من إنشاء دوال وأصناف عامة تدعم أي نوع بيانات، مما يُعزز إعادة استخدام الكود.

المكتبات القياسية STL

تقدم حاويات جاهزة مثل vector, map, set، وتُسهل الكثير من المهام.

الاستثناءات (Exceptions)

تُستخدم للتحكم بالأخطاء أثناء التشغيل باستخدام try وcatch.

إدارة الذاكرة

باستخدام new وdelete، مع إمكانية استخدام المؤشرات الذكية (smart pointers) في C++11 وما بعدها.
